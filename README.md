# Vinjector
Video banner and icon editor manipulation for promoting sites using a video

usage:
<pre>
  $ ./vinjector -f logo.gif -i icon.png -p video.mp4 -d 30 -o video -m 7.5 -x
</pre>

instalation ffmpeg (required before you used vinjector):
<pre>
  $ apt install ffmpeg
</pre>

# supported platform
1. Termux (android)
2. Linux
